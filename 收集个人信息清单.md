| --------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------ | -- |
| 友盟数据统计    | 包括但不限于设备序列号 / 手机号 / 设备 ID (DEVICE\_ID) / IMEI / Mac / androidID / IDFA / BSSID / ICCID / SN / OPENUDID / GUID / SIM 卡 IMSI / 地理位置信息或其他永久性或非永久性设备标识符（“设备标识符”）；设备平台和固件；ICCID；手机运营商；邮政编码、地域码、粗略定位等地理数据 | 用于统计 App 使用情况，方便 App 迭代更新                  | 查看 |
